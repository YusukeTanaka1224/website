# 公開手順と更新マニュアル（田中佑典 個人サイト）

コードを書く作業は一切ありません。所要時間は公開まで約15分です。
手順は「①GitHub」→「②Netlify」→「③管理画面の有効化」の3つだけです。

---

## ① GitHub にサイトのファイルを置く（5分）

GitHubは「サイトのファイルの保管場所」です。

1. https://github.com を開き、「Sign up」からアカウントを作成（メールアドレスとパスワードを登録）。
2. ログイン後、右上の「＋」→「New repository」をクリック。
3. Repository name に `website` と入力し、「Public」のまま「Create repository」を押す。
4. 次の画面で「uploading an existing file」という青い文字リンクをクリック。
5. お手元のZIPを解凍してできた **「site」フォルダの中身すべて**（index.html、css、js、data、admin、uploads）を、画面にドラッグ＆ドロップ。
   ※「site」フォルダごとではなく、**中身**を入れるのがポイントです。
6. 下の緑のボタン「Commit changes」を押す。

## ② Netlify で公開する（5分）

Netlifyは「保管したファイルをインターネットに公開してくれる」無料サービスです。

1. https://app.netlify.com を開き、「Sign up」→「**GitHub**」を選んでログイン（①のアカウントが使えます）。
2. 「Add new site」→「Import an existing project」→「GitHub」を選択。
3. 先ほど作った `website` を選ぶ。設定画面は**何も変えずに**「Deploy」を押す。
4. 1分ほどで公開されます。「Site configuration」→「Change site name」で
   サイト名を `yusuketanaka-vernacular` に変更すると、
   **https://yusuketanaka-vernacular.netlify.app** がサイトのURLになります。
   （もし使用済みと表示されたら `yusuketanaka-v` など少し変えてください）

## ③ 管理画面（自分で編集する仕組み）を有効にする（5分）

1. Netlifyのサイト管理画面で「**Identity**」タブ →「Enable Identity」を押す。
2. 同じ画面の「Identity」→「Configuration and usage」→ Registration を「**Invite only**」に変更（他人が勝手に登録するのを防ぎます）。
3. 「Identity」→「Services」→「**Git Gateway**」→「Enable Git Gateway」を押す。
4. 「Identity」タブ →「Invite users」→ ご自身のメールアドレスを入力して招待。
5. 届いたメールの「Accept the invite」を押すと、サイトが開きパスワード設定画面が出ます。パスワードを決めて完了。

以後、**https://（サイトURL）/admin/** がご自身専用の管理画面です。
ブックマークしておいてください。

## 日々の更新方法（管理画面の使い方）

管理画面を開くと「プロフィール」「原点（Story）」「論文リスト」「サイト設定」の項目が並んでいます。
クリック → 書き換え →「**Publish**」ボタン、で完了です。1〜2分で実際のサイトに反映されます。

- **顔写真を入れる**：「プロフィール」→「顔写真」欄でお手元の画像を選ぶと、自動でアップロードされProfileに表示されます（縦長の写真がきれいに収まります）。
- **原点（Story）を書く**：「原点（Story）」→ 本文を入力して Publish。冒頭の一文や、村の風景写真（任意）も添えられます。
- **論文を追加する**：「論文リスト」→ 論文欄の「＋」→ タイトル等を入力。
  「PDFファイル」欄でPDFを選ぶと「PDF ↓」リンクが、「CiNiiのURL」に論文ページのアドレスを貼ると「CiNii →」リンクが、それぞれ付きます。
- **プロフィールを直す**：「プロフィール」→ 文章を書き換えて Publish。経歴は空行で段落が分かれます。
- **noteの記事一覧**：サイト内の中継の仕組み（_redirectsファイル）で自動取得しているので、**何もしなくて大丈夫です**。「サイト設定」→「Newsの見せ方」で、サムネイル付きカード／タイトルのみの行、を切り替えられます。※将来noteのアカウント名を変えた場合だけ、GitHub上の「_redirects」ファイル内のnoteアドレスの書き換えが必要です。万一表示が止まった場合は、「記事の手動リスト」に入れればそちらが表示されます。

## Googleフォーム（連絡先）の設定

1. Googleフォームで連絡用フォームを作成し、「送信」ボタン → リンクのマーク → URLをコピー。
2. 管理画面の「サイト設定」→「GoogleフォームのURL」に貼り付けて Publish。
3. 同じ画面の「連絡フォームの見せ方」で、いつでも切り替えられます：
   - **ボタン**：黒い画面に静かなボタンだけが置かれ、押すとフォームが別画面で開く（サイトの雰囲気を保ちやすく、おすすめ）
   - **埋め込み**：ページ内にGoogleフォームがそのまま表示される（訪問者の手間は少ないが、フォームの白い画面が現れる）
   両方を実際に試して、お好みの方をお選びください。

## 困ったとき

- 編集しても反映されない → 1〜2分待ってからページを再読み込み（Ctrl/Cmd＋R）。
- 管理画面に入れない → /admin/ を開き「Login with Netlify Identity」→ 設定したメールとパスワード。
- パスワードを忘れた → ログイン画面の「Forgot password?」から再設定。
